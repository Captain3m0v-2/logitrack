# Interactive Analytics Dashboard

A modern, responsive dashboard with interactive charts and data visualization.

## Features

- **Interactive Charts**: Line and bar charts using Chart.js
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Filter Controls**: Change time periods and chart types
- **Animated Statistics**: Smooth transitions when data updates
- **Data Table**: Recent orders with status indicators
- **Notifications**: Feedback for user actions
